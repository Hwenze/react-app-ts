# react-app
使用webpack自行搭建react环境，使用react 16.4 + react-router 4.3 + typescript + mobx 实现antd-pro
\
使用node + mock 模拟数据

## 启动项目
```sh
安装依赖
$ npm install
启动模拟数据
$ node .\server\server.js
启动项目
$ npm run start
在浏览器中输入 http://localhost:8000
```
